# Kanal D Video Downloader

Merges the parts of a shows into one file, then download.

## Installation

```bash
git clone https://github.com/melihcaliskan/kanald-video-downloader
npm i
```

##Prerequisites
* node
* ffmpeg

## Usage

```bash
node index.js [show-link-1] [show-link-2]
```
